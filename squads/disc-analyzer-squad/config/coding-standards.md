# Coding Standards — disc-analyzer-squad

extends: core

## Padrões específicos do squad

- Pontuações DISC sempre como inteiros ou floats com 1 casa decimal
- Labels de perfil sempre em CAIXA ALTA: EXECUTORA, COMUNICATIVA, PLANEJADORA, ANALISTA
- Dimensões referenciadas pela letra maiúscula: D, I, S, C
- Outputs de laudo seguem o template definido em classify-disc-profile.md
